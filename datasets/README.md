# Datasets

Training and evaluation datasets for the Prosody Protocol project.

## Structure

Each dataset is stored in its own subdirectory with the following layout:

```
dataset-name/
├── metadata.json    # Dataset metadata (size, version, license, etc.)
├── entries/         # One JSON file per annotated entry
├── audio/           # WAV audio files referenced by entries
└── README.md        # Dataset-specific documentation
```
